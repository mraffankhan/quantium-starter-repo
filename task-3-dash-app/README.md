# Task 3: Basic Dash Application

## Purpose
This task involves creating a simple Dash web application to visualize the sales data.

## Features
-   **Data Loading**: Reads `processed_sales.csv`.
-   **Visualization**: Displays a basic Line Chart of Sales vs Date.
-   **No Interactivity**: This version only displays the static trend over time.

## How to Run
```bash
python app.py
```
